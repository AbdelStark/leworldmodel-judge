# RFC-005 — Benchmark and baselines

- **Status:** Accepted
- **Date:** 2026-04-23

## Decision
The benchmark is mandatory and is part of the product, not an afterthought.

## Required baselines
- terminal success only
- cumulative sparse reward
- heuristic progress proxy

## Required metrics
- early failure detection
- ranking quality
- optional calibration if uncertainty exists

## Consequence
No result should be discussed without showing the baseline comparison.
